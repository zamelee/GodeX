# GodeX Favicon Pack

包含：

- favicon.ico：多尺寸 ICO，适合传统浏览器/根目录使用
- favicon.svg：矢量图标版本
- favicon-16x16.png / 32x32 / 48x48 / 64x64 / 96x96 / 128x128 / 192x192 / 256x256 / 512x512
- apple-touch-icon.png：iOS / Safari 收藏图标，180x180
- site.webmanifest：PWA / Android 图标配置
- favicon-html-snippet.html：可直接复制到 `<head>` 的引用代码

推荐放置到网站 `public/` 或根静态目录。
