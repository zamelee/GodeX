# GodeX Complete Design Assets

本资源包汇总了本轮对话中生成的 GodeX 品牌与 UI 设计资产，可直接用于官网、README、文档站、应用 UI、favicon、CLI 展示。

## 目录结构

```text
00-preview/                 视觉预览页，打开 index.html 查看
01-logo-system/             第一批拆分 Logo 资源：PNG / SVG / App icons / exact crops
02-favicon/                 favicon.ico / favicon.svg / 16-512 PNG / apple-touch-icon / manifest
03-standalone-renders/      最新版单独渲染 Logo：横版、深色、竖版、图标、黑白
04-brand-boards/            品牌规范展示板、裁切预览板
05-ui-theme-guide/          UI 设计主题文稿、CSS Tokens、Tailwind Theme、favicon HTML 片段
```

## 推荐优先使用

- 官网 Header：`01-logo-system/svg/godex-logo-horizontal-gradient.svg`
- 深色 Hero：`01-logo-system/svg/godex-logo-horizontal-on-dark.svg`
- favicon：`02-favicon/favicon.ico` 或 `02-favicon/favicon.svg`
- Apple Touch Icon：`02-favicon/apple-touch-icon.png`
- App Icon：`01-logo-system/app-icons/godex-app-icon-1024.png`
- UI 主题规范：`05-ui-theme-guide/GodeX_UI_Design_Theme_Guide.md`
- CSS Tokens：`05-ui-theme-guide/godex-theme-tokens.css`
- Tailwind Theme：`05-ui-theme-guide/godex-tailwind-theme.ts`
