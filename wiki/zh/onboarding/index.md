# 入门指南

GodeX 是一个 OpenAI Responses API 网关，将 `/v1/responses` 请求转换为上游 Chat Completions API 调用，使任何 LLM 提供商都能驱动 Codex CLI 及其他 OpenAI 兼容客户端。

选择适合你角色的指南：

| 指南 | 读者 | 内容 | 阅读时间 |
|------|------|------|---------|
| [贡献者指南](./contributor-guide.md) | 有 TypeScript/JS 经验的新贡献者 | 环境搭建、架构深入、首次 PR、测试、调试 | ~45 分钟 |
| [架构师指南](./staff-engineer-guide.md) | 资深/首席工程师 | 架构设计、技术决策、系统边界、技术债务 | ~35 分钟 |
| [管理者指南](./executive-guide.md) | 工程副总裁/总监 | 能力地图、风险评估、成本模型、路线图 | ~15 分钟 |
| [产品经理指南](./product-manager-guide.md) | 产品经理、非工程利益相关者 | 功能、用户旅程、约束、常见问题 | ~15 分钟 |
