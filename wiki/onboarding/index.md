# Onboarding

GodeX is an OpenAI Responses API gateway that translates `/v1/responses` requests into upstream Chat Completions API calls, letting any LLM provider serve as a backend for Codex CLI and other OpenAI-protocol tools.

Choose the guide that matches your role:

| Guide | Audience | What You'll Learn | Time |
|-------|----------|-------------------|------|
| [Contributor Guide](./contributor-guide.md) | Engineers joining the project (TypeScript/JS background) | Setup, architecture deep-dive, first PR, testing, debugging | ~45 min |
| [Staff Engineer Guide](./staff-engineer-guide.md) | Staff/principal engineers | Architecture decisions, boundaries, failure modes, tech debt | ~35 min |
| [Executive Guide](./executive-guide.md) | VP/director-level engineering leaders | Capability map, risk assessment, cost model, roadmap alignment | ~15 min |
| [Product Manager Guide](./product-manager-guide.md) | Product managers, non-engineering stakeholders | Features, user journeys, constraints, FAQ | ~15 min |
